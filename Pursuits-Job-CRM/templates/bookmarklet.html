Bookmarklet

Save any job posting to Pursuits with one click.

{# The bookmarklet itself. When the user drags the link to their bookmarks bar (or right-clicks → Bookmark this Link), the browser stores the href as a bookmark. Clicking that bookmark on any page runs the JS in the href. What this one does: 1. Reads the current page's URL, title, and any selected text 2. URL-encodes them 3. Opens http://127.0.0.1:5000/new with those values as query params The /new route reads `url`, `company`, `role`, and `paste` query params and prefills the form (see the new() route). Combined with the LLM extract button, the workflow becomes: select the description on a posting, click the bookmarklet, click "Extract with Claude" on the form that opens, save. The IIFE wrapper is required — without it, declaring `var u` would leak into the host page's global scope and possibly conflict. #} {% set bookmarklet_js %}javascript:(function(){var u=encodeURIComponent(location.href),t=encodeURIComponent(document.title||''),s=encodeURIComponent((window.getSelection&&window.getSelection().toString())||'');window.open('{{ request.scheme }}://{{ request.host }}/new?url='+u+'&paste='+s+'&prefill_title='+t,'_blank');})();{% endset %}

Install

Drag this button to your bookmarks bar:

📎 Save to Pursuits

If your browser hides the bookmarks bar: Chrome / Edge — Ctrl+Shift+B · Firefox — right-click toolbar → Bookmarks Toolbar → Always Show · Safari — View → Show Favorites Bar

How to use

  1. 1.Open a job posting in your browser (LinkedIn, Indeed, a company careers page, anything).
  2. 2.Optional: select the job description with your mouse before clicking the bookmarklet — selected text comes along for extraction.
  3. 3.Click the 📎 Save to Pursuits bookmark.
  4. 4.A new tab opens to Pursuits with the URL prefilled.{% if llm_available %} If you selected text, hit "Extract with Claude" to fill the rest.{% endif %}
  5. 5.Review and Save.

Manual install

If drag-and-drop doesn't work in your browser, create a bookmark manually and paste this as the URL: 

{{ bookmarklet_js }}

Note

The bookmarklet points at {{ request.scheme }}://{{ request.host }} — the URL you're viewing right now. If you change the port your Flask app runs on, come back to this page and re-drag the link to update it.
